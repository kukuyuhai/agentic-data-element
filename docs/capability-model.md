# 能力模型（Capability Model）

本项目将 T/MIITEC 025-2024 定义的岗位能力抽象为**三维四层**的结构化模型，
用于机器可读、可组合、可评估地表达每一个数字员工的胜任力。

## 1. 三维模型

```
                  ┌────────────────────┐
                  │  专业知识 knowledge │
                  ├────────┬───────────┤
                  │ 基础知识 │ 专业知识 │
                  └────────┴───────────┘
                           │
       ┌───────────────────┼────────────────────┐
       ▼                   ▼                    ▼
┌────────────┐     ┌──────────────┐     ┌────────────┐
│ 技术技能    │     │  岗位能力总分 │     │ 工程实践    │
│ skill      │     │  Capability   │     │ practice   │
├────┬───────┤     └──────────────┘     ├────┬───────┤
│基本│ 专业   │            ▲              │体系│复杂问题│
└────┴───────┘            │              └────┴───────┘
                    等级权重加权求和
```

## 2. 四层结构

| 层级 | 描述 | 举例 |
|---|---|---|
| L1 · 维度 | 三维 | 专业知识 / 技术技能 / 工程实践 |
| L2 · 子维度 | 每维度 1 ~ 2 项 | 基础知识 / 专业知识 |
| L3 · 能力条目 | 具体能力点 | "掌握资产负债表、损益表、现金流量表等财务报表的解读" |
| L4 · 标准锚点 | 与 T/MIITEC 025-2024 章节精确对齐 | `§5.1.10 a)` |

## 3. 能力等级与权重

| 等级 | 档次 | knowledge | skill | practice |
|---|---|---|---|---|
| 1 ~ 3 | 初级 | 0.70 | 0.25 | 0.05 |
| 4 ~ 6 | 中级 | 0.50 | 0.25 | 0.25 |
| 7 ~ 9 | 高级 | 0.20 | 0.30 | 0.50 |

> 初级偏"知识"、中级偏"独立作业"、高级偏"工程实践与体系搭建"。

## 4. 岗位能力分（示例算法）

```python
def capability_score(knowledge, skill, practice, level="senior"):
    """
    knowledge / skill / practice: 0~100 的评分
    level: junior | mid | senior
    """
    weights = {
        "junior": (0.70, 0.25, 0.05),
        "mid":    (0.50, 0.25, 0.25),
        "senior": (0.20, 0.30, 0.50),
    }[level]
    return sum(w * s for w, s in zip(weights, (knowledge, skill, practice)))
```

## 5. 数字员工能力向量

每个 agent 文件的 front-matter 中已声明 `level`，
配合 `roster.json` 的 `capability_dimensions`、`levels`，
即可构成完整的能力向量（供上层评价系统消费）。

```yaml
# 示例：DAM-07 数据资产评估计价师
id: DAM-07
level: 高级          # senior
knowledge:
  base:    ["资产评估准则", "国际评估准则", "金融基础"]
  advanced: ["数据资产估值方法（成本/收益/市场）"]
skill:
  base:    ["Python 建模", "SQL"]
  advanced: ["蒙特卡洛模拟", "版权/无形资产评估工具"]
practice:
  - 主导过省级数据交易所试点估值
  - 输出过 20+ 份评估报告
```

## 6. 与其它模型的映射

| 本项目 | 岗位能力标准（人社部） | DCMM（GB/T 36073） |
|---|---|---|
| 三维模型 | 知识 / 技能 / 能力 | 数据战略 / 治理 / 管理 / 应用 |
| 能力等级 1-9 | 五级职业等级 | 五级成熟度（初始 → 优化） |
| 标准锚点 §5.X.X | 对应条款 | 相关域条款 |
