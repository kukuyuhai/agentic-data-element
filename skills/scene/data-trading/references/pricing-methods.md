# 数据产品定价方法论

> 数据定价是交易中最难的一环。核心难题：数据可复用（非竞争性）、价值因场景而异、时效性强。

---

## 一、三大定价方法

### 1. 成本法

**公式**：数据产品价格 = 全链路成本 × (1 + 合理利润率)

**成本构成**：
| 成本项 | 具体内容 |
|--------|---------|
| 数据采集成本 | 自产系统的开发/运维分摊；外购数据的采购成本 |
| 数据处理成本 | 清洗、标注、脱敏、格式转换等加工成本 |
| 存储成本 | 服务器/云存储租赁费 |
| 合规成本 | 合规审查费、安全评估费、脱敏改造费 |
| 维护成本 | 持续更新、数据质量监测 |
| 管理分摊 | 人员薪酬、管理费 |

**适用场景**：首次定价、缺少可比交易案例、内部使用或自用增效类数据产品

**局限**：完全忽略了数据的使用价值（同样的成本投入，对不同需求方价值差异巨大）

---

### 2. 市场法

**方法**：参考同类数据产品的市场价格区间

**数据来源**：
- 各交易所已挂牌产品的公开报价
- 行业报告中的数据交易行情
- 公开招投标中的数据采购价格

**调整因子**：
| 因子 | 说明 | 调整方向 |
|------|------|---------|
| 数据质量 | 完整度、准确度、时效性 | 质量越高，溢价越高 |
| 应用场景 | 金融风控 > 科研学术 | 场景价值越高，溢价越高 |
| 独占性 | 独家 vs 多家供应 | 独占大幅溢价 |
| 覆盖范围 | 全国 vs 单区域 | 范围越广，溢价越高 |
| 更新频率 | 实时 > T+1 > 月度 | 更新越快，溢价越高 |
| 合规等级 | 高合规 > 基础合规 | 合规等级越高，溢价越高 |

**适用场景**：有可比交易案例的成熟数据类型（如金融征信数据、气象数据、电力数据）

**局限**：数据交易市场尚不成熟，可比案例少；数据异构性强，很难直接对标

---

### 3. 收益法

**方法**：数据产品未来预期收益的现值

**公式**：数据产品价值 = Σ(各期预期收益 / (1 + 折现率)^t)

**收益测算路径**：
- 直接收入：对外销售/许可的收入
- 间接收入：通过数据产品改善业务带来的增收/节本
- 分成收入：按需求方使用后产生的收益比例分成

**折现率参考**：
- 低风险场景（公共数据、成熟业务）：6%-10%
- 中等风险（企业数据、成长期业务）：10%-15%
- 高风险（创新数据、试水场景）：15%-25%

**适用场景**：有明确商业模式和收入预期的数据产品、收益分成模式

**局限**：收益预测主观性强、不确定性大；折现率选择争议大

## 二、贵阳数据产品交易价格计算器

### 使用逻辑

由国家发改委价格监测中心指导、贵阳大数据交易所推出的全国首个数据产品定价工具。

**输入参数**：
1. **行业类别**：金融/医疗/制造/零售/交通/政务/农业/教育/气象/能源/其他
2. **应用场景**：风控/营销/研发/运营/决策/监管/训练/其他
3. **数据质量等级**：A级(高) / B级(中) / C级(基础)
4. **开发成本**：总投入（含采集、清洗、标注、脱敏、存储等）
5. **数据规模**：记录数/数据量
6. **更新频率**：实时/T+1/周度/月度/静态
7. **覆盖范围**：全国/多省/单省/单市
8. **独占性**：独家/多家/公开

**输出**：具有公信力的参考价格区间（上限/中位/下限）

### 本地替代方案

如果无法直接使用贵阳价格计算器，可按以下简化公式估算：

```
参考价格 = 全链路成本 × 行业系数 × 质量标准系数 × 场景系数 + 市场比较溢价/折价
```

- **行业系数**：金融 1.5、医疗 1.3、政务 0.8、气象 1.2、交通 1.1、其他 1.0
- **质量标准系数**：A级 1.5、B级 1.0、C级 0.7
- **场景系数**：风控/训练 1.5、营销 1.2、研发 1.1、运营/决策 1.0

**注意**：上述系数为经验参考值，实际交易以供需双方协商为准。

## 三、三种定价交易模式

| 模式 | 说明 | 适用场景 | 典型收费方式 |
|------|------|---------|-------------|
| **明码标价** | 供方公布固定价格，需方直接下单 | 标准化数据产品 | 一次性/按次/按周期 |
| **询价议价** | 供需双方协商确定价格 | 定制化/大额交易 | 灵活协商 |
| **公开竞价** | 多家需方竞价，价高者得 | 稀缺/独占数据 | 竞价 |
| **收益分成** | 不收取/少收取前期费用，按需方收益分成 | 创新应用、效果不确定 | 按收入/利润比例 |

---

## 四、定价中的常见误区

1. **"入表金额 = 交易价格"**：入表按历史成本，交易按市场价值，两者通常差距巨大。
2. **"评估价值 = 交易价格"**：第三方评估报告给出的价值，不等于市场愿意接受的价格。评估价值偏高是普遍现象。
3. **"独占使用权 = 买断所有权"**：数据交易通常仅转让使用权（许可），不转让所有权。价格应反映使用范围和期限。
4. **"一次定价，终身不变"**：数据具有强时效性，价格随时间衰减，应约定调整机制。
