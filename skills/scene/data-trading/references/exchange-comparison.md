## 各交易所流程详解

### 1. 贵阳大数据交易所

**定位**：全国首家数据流通交易场所，"让数据交易像网购一样便捷"

**入驻流程**（4步）：
1. 主体登记 → 获得数据商凭证
2. 数据产品上架 → 按统一标准在平台登记
3. 定价 → 协商定价 / 自动定价 / 评估定价（含价格计算器辅助）
4. 交易撮合 → 线上+线下，"交易大厅"实时撮合

**配套工具**：
- **数据要素登记 OID 服务平台**：基于国家 OID 注册中心授权，提供产品登记、资产登记、交易登记
- **数据产品交易价格计算器**：全国首个，由国家发改委价格监测中心指导
- **区块链交易存证**：全流程上链，可追溯

**适用场景**：对定价有不确定性、首次进入数据交易市场的企业；适合中小数据商

---

### 2. 上海数据交易所

**定位**：最规范化的数据交易场所，"不合规不挂牌，无场景不交易"

**流程（三步走）**：
1. **数据产品登记** → 信息存证、产品溯源、权益保护、集聚展示、场景探索
2. **挂牌上架** → 补充挂牌信息 + 提交合规评估报告 + 质量评估报告 → 获得挂牌证书
3. **交易执行** → 签约 → 交付 → 结算 → 颁发凭证

**关键规则**：
- 挂牌后 12 个月内未发生场内交易 → 自动摘牌
- 撤销挂牌条件：数据产品被没收/征收/收回、法律文书导致权益消灭
- 交易凭证可被撤销（主体不合法/程序不合规/虚假材料）

**适用场景**：合规要求高、交易金额大、有明确应用场景的数据产品

---

### 3. 深圳数据交易所

**定位**：跨境数据交易先行者

**特色能力**：
- 率先探索数据跨境交易机制
- RWA（Real World Asset）数据资产代币化方向
- 大湾区数据互联互通

**入驻要求**：与上海类似，增加跨境专项审查

**适用场景**：涉及跨境数据流通、大湾区业务、RWA 探索型企业

---

### 4. 北京国际大数据交易所

**定位**：国有数据资源交易主阵地

**特色方向**：
- 公共数据授权运营产品交易
- 国有数据资源进场
- 合规审查门槛最高

**入驻流程**：主体审核 → 数据来源审查 → 合规评估 → 挂牌 → 撮合

**适用场景**：央企/国企数据资源、公共数据授权运营、政务数据产品化

---

### 5. 西部数据交易中心（重庆）

**定位**：西部产业数据交易枢纽，2026年上架产品数全国第一

**特色能力**：
- "1+6"规则体系：1 个总则 + 6 个细则（安全、主体、合规、定价、交付、结算）
- 汽摩、电子信息等产业数据专区
- 《重庆市数字经济促进条例》明确支持

**适用场景**：西部企业、制造业/汽摩/电子信息产业数据、公共数据融合利用

## 交易所选择决策树

```
第一步：看数据来源
  ├── 公共数据/国有数据 → 优先考虑北京或贵阳
  ├── 企业数据 → 根据行业和地区选择
  └── 涉及跨境 → 优先考虑深圳

第二步：看交易特征
  ├── 首次交易、拿不准定价 → 贵阳（价格计算器）
  ├── 合规要求极高 → 上海（最规范）
  ├── 产业数据大宗交易 → 西部/贵阳
  └── 小批量快速交易 → 均可，贵阳线上化程度高

第三步：看地理因素
  ├── 华南/大湾区 → 深圳
  ├── 华东 → 上海
  ├── 西南 → 贵阳/西部(重庆)
  └── 华北 → 北京

第四步：看附加服务需求
  ├── 需要 OID 登记 + 价格计算 → 贵阳
  ├── 需要更广的供需对接 → 西部（数商最多）
  └── 需要跨境 → 深圳
```
