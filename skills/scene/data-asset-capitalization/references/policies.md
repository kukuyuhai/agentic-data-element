# 数据资产入表核心政策法规

## 一、核心文件

### 《企业数据资源相关会计处理暂行规定》
- **发文机关**: 财政部
- **文号**: 财会〔2023〕11号
- **施行日期**: 2024年1月1日
- **核心内容**: 明确企业数据资源在满足条件时可确认为无形资产或存货，规范初始计量、后续计量、列报与披露要求

### 关键条文速查

**适用范围（第二条）**: 企业按照企业会计准则相关规定确认为无形资产或存货等资产类别的数据资源，以及企业合法拥有或控制的、预期会给企业带来经济利益的、但由于不满足企业会计准则相关资产确认条件而未确认为资产的数据资源，其相关会计处理适用本规定。

**确认条件（参阅《企业会计准则第6号——无形资产》）**:
1. 与该资产有关的经济利益很可能流入企业
2. 该资产的成本能够可靠地计量

**初始计量（第四条）**: 企业通过外购方式取得的数据资源，其成本包括购买价款、相关税费、以及数据标注、整合、分析或可视化等服务所发生的有关支出。企业通过加工形成的数据资源，其成本包括直接发生的加工成本以及使该项数据资源达到预定用途所发生的必要支出。

**后续计量（第六条/第七条）**:
- 使用寿命有限的无形资产：在使用寿命内系统合理摊销
- 使用寿命不确定的无形资产：不摊销，每年进行减值测试
- 存货：不计提摊销，期末按成本与可变现净值孰低计量

**列报与披露**: 
- 资产负债表中"无形资产"项目下设"其中：数据资源"子项目
- "存货"项目下设"其中：数据资源"子项目
- 附注中披露数据资源的类别、增减变动、摊销方法、减值信息等

## 二、配套制度体系

### 数据产权"三权分置"
- **依据**: 中共中央、国务院《关于构建数据基础制度更好发挥数据要素作用的意见》（"数据二十条"，2022年12月）
- **核心**: 建立数据资源持有权、数据加工使用权、数据产品经营权"三权分置"的产权运行机制
- **与入表的关系**: 入表企业必须证明其对数据资源拥有合法权利（持有权、使用权、经营权之一），这是确认数据资产的前提

### 数据资产评估
- **依据**: 《数据资产评估指导意见》（中国资产评估协会，2023年9月）
- **三大方法**: 成本法、收益法、市场法
- **成本法适用场景**: 企业自用数据资产、成本记录完整、无活跃交易市场时首选
- **收益法适用场景**: 数据产品有明确商业化路径、可预测未来收益
- **市场法适用场景**: 存在可比交易案例、活跃的数据交易市场

### 相关法律法规
- 《中华人民共和国数据安全法》（2021年9月施行）
- 《中华人民共和国个人信息保护法》（2021年11月施行）
- 《中华人民共和国网络安全法》（2017年6月施行）
- 《企业会计准则第6号——无形资产》
- 《企业会计准则第1号——存货》

## 三、数据资产入表前提条件

入表前必须确认以下5个条件同时满足：

1. **合法性**: 数据来源合法，符合《数据安全法》《个人信息保护法》要求
2. **权属性**: 企业合法拥有或控制该数据资源（三权分置中至少有一权）
3. **可辨认性**: 数据资源可与企业其他资产明确区分，可单独识别和管理
4. **经济利益**: 预期能为企业带来未来经济利益（可自用增效或对外交易）
5. **成本可计量**: 取得或形成该数据资源的成本能够可靠计量

## 四、数据资产分类判定逻辑

```
企业拥有/控制的数据资源
├── 用于内部使用或对外提供服务
│   └── 满足无形资产确认条件 → 确认为"无形资产-数据资源"
├── 日常活动中持有、最终用于出售
│   └── 满足存货确认条件 → 确认为"存货-数据资源"
└── 不满足资产确认条件
    └── 不在资产负债表中确认，附注中自愿披露
```
