# 数据资产会计处理详解

## 一、初始计量 - 成本归集

### 外购数据资产

```
数据资产成本 = 购买价款 + 相关税费 + 使资产达到预定用途的合理必要支出
```

**可资本化的支出示例**:
- 数据购买价款（含授权使用费）
- 数据格式转换、清洗、去重的直接人工和费用
- 数据标注、分类的直接成本
- 数据质量检测与校验费用
- 数据整合、分析、可视化的直接服务费用

**不可资本化的支出**（费用化处理）:
- 数据基础设施的日常运维费用
- 数据更新维护的常态化支出
- 一般管理费用
- 员工培训费用

### 自行加工形成的数据资产

**"研究阶段"支出 → 全部费用化**
研究阶段特征：探索性、无明确应用目标，如：
- 数据资产入表的可行性研究
- 数据资源盘点和初步筛选
- 技术路线探索和验证

**"开发阶段"支出 → 满足5个条件后资本化**
开发阶段特征：有明确应用目标、技术可行性已验证，如：
- 数据产品的具体开发
- 数据模型的构建和训练
- 数据库/数据平台的具体建设

**开发支出资本化5个条件**（全部满足）:
1. 完成该数据资产以使其能够使用或出售在技术上具有可行性
2. 具有完成该数据资产并使用或出售的意图
3. 数据资产产生经济利益的方式能够证明其有用性
4. 有足够的技术、财务资源和其他资源支持以完成开发并有能力使用或出售
5. 归属于该数据资产开发阶段的支出能够可靠地计量

### 成本归集示例

| 成本项目 | 资本化/费用化 | 说明 |
|---------|:---------:|------|
| 外部数据采购费 | 资本化 | 构成数据资产直接成本 |
| 数据标注人工 | 资本化 | 开发阶段直接人工 |
| 服务器折旧 | 按比例分摊 | 按实际使用量分摊资本化 |
| 可行性研究费 | 费用化 | 研究阶段支出 |
| 日常运维费 | 费用化 | 不属于达到预定用途的必要支出 |
| 数据合规审查费 | 资本化 | 使数据资产可合法使用的必要支出 |

## 二、后续计量

### 无形资产 - 数据资源

**摊销**:
- 使用寿命有限：在使用寿命内按直线法（或其他合理方法）系统摊销
- 使用寿命确定因素：数据的时效性、技术迭代速度、合同限制、法律保护期限
- 摊销年限常见区间：3-10年（需根据数据特性和业务模式合理估计）
- 使用寿命不确定：不摊销，每年进行减值测试

**摊销分录**:
```
借：管理费用/主营业务成本——数据资源摊销
  贷：累计摊销——数据资源
```

**减值测试**（至少每年末进行）:
- 减值迹象：数据时效性丧失、法律法规变化导致不可用、技术进步使数据贬值、商业模式变化
- 减值损失 = 账面价值 - 可收回金额
- 减值分录：
```
借：资产减值损失——数据资源
  贷：无形资产减值准备——数据资源
```

### 存货 - 数据资源

- **不摊销**
- 期末按"成本与可变现净值孰低"计量
- 可变现净值 = 估计售价 - 估计销售费用 - 相关税费
- 跌价准备计提：
```
借：资产减值损失——数据资源
  贷：存货跌价准备——数据资源
```

## 三、处置与终止确认

**出售/转让**:
```
借：银行存款（实际收到的金额）
    累计摊销——数据资源
    无形资产减值准备——数据资源
  贷：无形资产——数据资源（账面原值）
      资产处置损益（差额，或在借方）
```

**报废**:
```
借：营业外支出
    累计摊销——数据资源
    无形资产减值准备——数据资源
  贷：无形资产——数据资源
```

## 四、报表列报

### 资产负债表列示

```
无形资产：
  其中：数据资源                                XXX
存货：
  其中：数据资源                                XXX
```

### 附注披露要点（强制披露）

1. 数据资源的类别（按业务类型/数据来源/使用方式等分类）
2. 各类数据资源期初、期末账面原值
3. 各类数据资源期初、期末累计摊销额
4. 各类数据资源期初、期末减值准备累计金额
5. 数据资源使用寿命的确定依据
6. 所有权或使用权受到限制的数据资源说明

### 附注披露要点（自愿披露）

1. 未确认为资产的数据资源情况
2. 数据资源的应用场景（内部管理、产业数字化、数字产业化）
3. 数据资源的业务模式
4. 数据资源的安全保护措施
5. 重大数据资源交易事项
