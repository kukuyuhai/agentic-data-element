# ⚖️ regulation-lookup

## When to Use

当出现以下场景时调用本技能：

- 输出涉及**合规判断**（如"这个数据能不能出境""采集是否合法"）
- 起草或审核**合同条款**（如数据许可、产权转让协议）
- 涉及**财务处理**（如数据资产入表科目选择、摊销年限）
- 需要在结论中引用法规条文作为依据
- 判断某行为是否触发**一票否决**或**申报义务**

本技能与 `standard-lookup` 互补：standard-lookup 查团体/国家标准（技术规范），regulation-lookup 查法律法规（合规依据）。两者经常配合使用。

## 工作原则

1. **法规原文不可改写**：返回条文必须是法规原文，概括性转述须标注"概述"
2. **效力层级标注**：法律 > 行政法规 > 部门规章 > 规范性文件，引用时需标注层级
3. **时效性检查**：注意法规是否已被修订或废止，引用时标注施行日期
4. **区分义务与禁止**："应当"为义务性规范，"不得"为禁止性规范，"可以"为授权性规范——引用时保留原措辞

## 法规索引

| 法规名称 | 层级 | 施行日期 | 核心覆盖范围 |
|---------|------|---------|------------|
| 数据二十条 | 规范性文件 | 2022-12-19 | 三权分置、数据要素流通、收益分配（第一~九条） |
| 数据安全法 | 法律 | 2021-09-01 | 数据分级分类保护、安全管理制度、跨境传输（第二十一~二十七条） |
| 个人信息保护法 | 法律 | 2021-11-01 | 处理规则、敏感信息、跨境提供、权利保障（第十三~三十九条） |
| 网络安全法 | 法律 | 2017-06-01 | 网络数据保护、关键信息基础设施（第二十一~三十九条） |
| 财会〔2023〕11号 | 部门规章 | 2024-01-01 | 数据资源会计确认、初始/后续计量、列报披露（第二~八条） |
| 数据出境安全评估办法 | 部门规章 | 2022-09-01 | 出境申报条件、评估流程、监管要求（第四~九条） |
| 《个人信息出境标准合同办法》 | 部门规章 | 2023-06-01 | 标准合同适用条件与备案要求 |

## Inputs

- **查询关键词**（string）：用户想检索的法规主题，如"数据出境""个人信息敏感信息""数据资产入表条件"
- **可选·法规限定**（string）：用户指定只查某部法规，如"数据安全法"
- **可选·合规场景**（string）：当前场景类型（入表/交易/共享/出境），用于优先返回相关条款

## Steps

### 步骤一：解析查询意图

1. 从用户输入中提取关键词，判断查询类型：
   - **合法性判断**："这样做合法吗" → 查数据安全法、个保法的禁止性条款
   - **义务判断**："需要申报吗" → 查数据出境安全评估办法、个保法出境条款
   - **权利归属**："数据归谁" → 查数据二十条三权分置
   - **会计处理**："怎么入账" → 查财会〔2023〕11号
   - **合同条款**："合同怎么写" → 查相关法规的权利义务规定，转交 `trading/contract-clause-library`

### 步骤二：检索法规库

1. 在法规索引中按关键词匹配，定位到具体法规与条文
2. 提取条文原文（≤ 300 字），保留"应当/不得/可以"等关键措辞
3. 如果同一主题涉及多部法规（如"数据出境"同时涉及数据安全法、个保法、出境评估办法），全部列出并标注层级关系

### 步骤三：标注义务类型

对每条检索结果标注义务类型：
- **义务性**（"应当"）：必须执行，不执行即违规
- **禁止性**（"不得"）：不可触碰，触碰即违法
- **授权性**（"可以"）：赋予权利，可选择是否行使
- **程序性**：规定操作流程与时限

### 步骤四：输出引用

将检索到的条文以标准格式输出供引用：

```
[《数据安全法》第三十一条] 关键信息基础设施的运营者在中华人民共和国境内运营中收集和产生的个人信息和重要数据应当在境内存储……
```

如涉及标准交叉引用，建议同时调用 `foundation/standard-lookup`。

## 常见问法与处理策略

| 用户问法 | 处理策略 |
|---------|---------|
| "数据出境要不要申报？" | 查数据出境安全评估办法第四条 + 个保法第三十八条，列出触发条件 |
| "三权分置是什么？" | 查数据二十条，返回三权定义与分离原则原文 |
| "个人敏感信息怎么处理？" | 查个保法第二十八~三十二条，返回敏感信息处理规则 |
| "入表需要什么法规依据？" | 查财会〔2023〕11号第二条适用范围 + 数据二十条权属条款 |
| "数据采集最小必要原则在哪？" | 查个保法第六条 + 数据安全法，返回"最小必要"相关条文 |
| "不申报出境有什么后果？" | 查数据安全法 + 个保法法律责任章节，返回处罚条款 |
| "这个条款有没有标准依据？" | 转交 `foundation/standard-lookup` 查对应标准 |
