# 🏛️ credit-rating-committee

信用评级委员会流程管理工具。由数据信用评价机构（`dam-11`）使用，依据 T/MIITEC 025-2024 §5.1.11，规范信用评级的**初评 → 复核 → 上会 → 完稿**全流程及其卡点管控，含委员回避制度、异议处置规则、投票表决机制。确保评级过程独立、公正、可追溯。

## When to Use

- 对新评级对象启动首次信用评级流程。
- 对现有评级对象进行**定期重评**（年度/半年度）。
- 被评对象**申请复评**或**提出异议**。
- 评级模型或流程发生重大变更后首次使用。

## 工作原则

1. **程序正义**：评级流程的每一节点有明确的输入/输出和责任人。
2. **独立性**：委员与被评对象有利益关系的须回避，委员独立发表意见不受干预。
3. **充分讨论**：存在分歧时不急于投票，先充分讨论和补充信息。
4. **多数表决**：上会阶段采用 2/3 多数表决机制。
5. **全程可溯**：每个环节的决策记录留档，形成完整的评级档案。

## 一、评级流程四阶段

### 阶段一：初评

**责任人**：分析师（由 `dam-11` 指定）。

**工作内容**：

1. 收集被评对象的基础材料（营业执照、财务报表、数据产品清单等）。
2. 获取数据质量评估结果（来自 `dq-6d-check`）。
3. 执行红线审查（来自 `credit-red-lines`）。
4. 使用 `credit-scorecard-builder` 生成初步评分和等级。
5. 结合定量评分和定性信息，撰写初评报告草案。
6. 分析师签字确认，提交复核。

**输出**：初评报告草案 + 评分卡结果。

**卡点**：
- 材料不齐备 → 退回要求补交，暂停计时。
- 红线触发 → 终止评级流程，出具一票否决通知。

### 阶段二：复核

**责任人**：复核组（至少2人，其中之一为复核组长，不得为原分析师）。

**工作内容**：

1. 独立审阅初评报告草案。
2. 独立重算关键评分环节，验证评分卡应用正确性。
3. 审阅证据材料的充分性和可靠性。
4. 对初评结论提出质疑和修改建议。
5. 形成复核意见（同意/修改后同意/不同意）。
6. 若有修改建议，分析师修改后复核组再次审阅。

**输出**：复核意见 + 修订后的评级报告草案。

**卡点**：
- 复核组与分析师意见不一致 → 记录分歧点，双方陈述理由，带入上会讨论。
- 证据材料存在重大缺陷 → 要求补充材料，暂停流程。

### 阶段三：上会（委员会评审）

**责任人**：评级委员会（由 `dam-11` 组织，委员人数为奇数，不少于5人）。

**工作流程**：

1. **会前准备**（会前3个工作日）：
   - 向每位委员发送会议通知和材料包（初评报告、复核意见、分歧点清单）。
   - 委员确认是否需回避（回避原则见下文）。
   - 需回避委员不参会、不阅件、不参与后续任何环节。

2. **会上陈述**（30分钟）：
   - 分析师陈述评级要点和主要结论（15分钟）。
   - 复核组长陈述复核意见（5分钟）。
   - 被评方代表陈述（如有邀请，10分钟）。

3. **委员讨论**（30-45分钟）：
   - 委员自由提问和讨论。
   - 重点讨论：评分争议点、定性调整重大事项、风险关注点。
   - 主任委员主持讨论，确保议题聚焦、时间可控。

4. **投票决议**（15分钟）：
   - 投票表决议题：确认评级等级。
   - 投票方式：记名投票（公开委员姓名和表决意见）。
   - 通过标准：2/3 及以上赞成票。
   - 未通过：返回初评修订，或由主任委员提请专家复议。

**输出**：委员会决议（含投票结果和委员签字）。

**卡点**：
- 出席委员人数不足5人 → 延期上会。
- 投票未达 2/3 → 返回修改后重新上会。
- 出现新的重大问题 → 暂停会上讨论，补充调查后再上会。

### 阶段四：完稿

**责任人**：分析师（在原报告基础上根据委员会决议修改）。

**工作内容**：

1. 根据委员会决议修订评级报告。
2. 在被评方未提异议的前提下，完成终稿定稿。
3. 使用 `credit-report-drafter` 生成正式报告。
4. 将正式报告提交 `dam-11` 负责人签发。
5. 归档：评级报告、决议、投票记录、工作底稿全部存档。

**输出**：签发版正式信用评级报告。

**卡点**：
- 委员会要求修改的内容未充分体现 → 退回重新修改。
- 发现终稿与决议不一致 → 退回澄清。

## 二、回避制度

### 需回避的情形

| 序号 | 情形 | 具体判断标准 |
|------|------|--------------|
| 1 | 利益冲突 | 委员本人或其直系亲属在被评对象中拥有股权（任何比例） |
| 2 | 雇佣关系 | 委员本人在被评对象或其关联公司担任任何职务（含顾问） |
| 3 | 商业关系 | 委员所在单位与被评对象存在重大商业合作（> 年收入10%） |
| 4 | 亲属关系 | 委员的直系亲属在被评对象担任关键岗位（董事、高管、核心技术人员） |
| 5 | 其他重大利害关系 | 可能影响独立判断的其他关系，由主任委员裁定 |

### 回避程序

1. 委员收到会议通知后，自查是否需回避。
2. 如需回避，在会前2个工作日内向主任委员提交回避申请。
3. 主任委员确认回避，调整会议安排。
4. 隐瞒或未主动回避的，事后发现将取消委员资格并追溯审查此前参与的评级结论。

## 三、异议处置

### 被评方异议

被评方收到评级结果通知后，如对结论有异议，可在 **15个工作日** 内提出书面申诉。

| 异议类型 | 处理程序 | 处理时限 |
|----------|----------|----------|
| 对评分计算的异议 | 复核组重新核验评分卡应用，给出书面答复 | 10个工作日 |
| 对事实证据的异议 | 被评方提供新的反证材料，分析师核实后重新提交复核 | 20个工作日 |
| 对评级结论的整体异议 | 重组一个不含原评审委员的复议委员会重新评审 | 30个工作日 |
| 对其他程序性问题的异议 | 由 `dam-11` 负责人审定 | 15个工作日 |

**异议后结果**：
- 异议成立 → 修改评级结论或等级。
- 异议不成立 → 维持原结论，出具书面驳回理由。
- 终裁结论不可再议。

### 委员异议

委员在投票中投反对票的，须书面说明反对理由并记录在案。反对理由存档于评级档案，不影响决议的法定效力。

## Steps

1. **启动评级流程**：确认评级类型，指定分析师和复核组成员。
2. **准备初评材料**：分析师收集材料、执行评分卡、起草初评报告。
3. **复核审核**：复核组独立审阅，形成复核意见。
4. **材料准备与回避确认**：向委员分发材料，确认回避委员名单。
5. **召开评级委员会会议**：陈述 → 讨论 → 投票决议。
6. **形成委员会决议**：若通过，明确等级和主要依据；若不通过，说明原因并指示下一步。
7. **起草异议答复**（如适用）：若被评方提出异议，按对应程序处理。
8. **完稿与签发**：分析师最终定稿，负责人签发，档案归档。
9. **标注标准溯源**：每个流程节点标注 T/MIITEC 025-2024 §5.1.11 条款编号。

## 常见问法与处理策略

| 问法 | 处理策略 | 注意事项 |
|------|----------|----------|
| "委员人数不够5人怎么办" | 延期上会，直到至少5人可出席；不可降低出席人数门槛或找替代委员应急 | 人数不足的会议无效 |
| "有委员应该回避但不主动回避" | 其他委员或分析师发现后向主任委员举报；确认后取消该委员的投票资格，追溯已投决议 | 故意不回避属严重违规 |
| "被评方对评级结果不满意，多次申诉" | 一次评级只受理一次正式申诉；终裁结论不可再议；多次申诉须有新增的实质证据才予受理 | 避免恶意拖延和消耗资源 |
| "初评和复核意见差异很大怎么办" | 差异写入复核意见中，带入上会讨论；由委员会裁决采纳哪个意见或折中 | 不要刻意消除分歧 |
| "投票 3-2 算不算通过" | 不算！2/3 即5人中需要至少4票赞成才通过；低于4票即为不通过 | 投票规则须在全流程开始前确认 |
