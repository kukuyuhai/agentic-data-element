# 💵 数据资产收益法（DCF）估值计算

## When to Use

当数据资产评估需要采用收益法（折现现金流模型 DCF）进行估值计算时使用本技能。核心目标：依据 GB/T 42129-2022 §7.2 及《数据资产评估指导意见》，通过现金流预测、折现率选取、折现计算与敏感性分析，输出数据资产的收益法估值结论。

典型触发场景：
- 数据资产评估报告需要收益法结论
- 数据资产有明确的商业化路径和可预测的未来收益
- 交易定价、出资入股、质押融资等场景的估值支撑
- 成本法和市场法估值结论的交叉验证

## 工作原则

1. **收益可预测性**：收益法仅在数据资产有明确商业化路径、未来收益可合理预测时使用。
2. **假设审慎性**：现金流预测的假设须审慎保守，避免过度乐观。
3. **折现率匹配**：折现率须反映数据资产特有风险，与现金流口径匹配（税前/税后）。
4. **敏感性必做**：必须对折现率、收入、期限等关键变量进行敏感性分析。
5. **准则溯源**：每条结论附 `§7.2.X` 标准溯源标注。

## DCF 估值方法步骤

### 第一步：现金流预测

**1.1 收入预测**

| 收入类型 | 预测方法 | 关键假设 |
|---------|---------|---------|
| 对外销售 | 单价 × 预计销售量 | 价格趋势、市场需求 |
| 对外服务 | 服务费率 × 服务量 | API调用次数、订阅数 |
| 内部使用 | 成本节约法 | 替代方案成本对比 |
| 授权许可 | 许可费率 × 使用规模 | 授权范围、期限 |

**1.2 成本预测**

| 成本类型 | 预测方法 | 说明 |
|---------|---------|------|
| 维护成本 | 历史成本 + 增长率 | 数据更新、质量维护 |
| 运营成本 | 按收入比例估算 | 通常为收入的 15%-30% |
| 技术更新成本 | 按计划投资 | 平台升级、技术迭代 |
| 相关税费 | 法定税率 | 增值税、所得税等 |

**1.3 净现金流计算**
```
净现金流 = 预测收入 - 预测成本 - 相关税费 - 资本性支出
```

### 第二步：折现率选取

**2.1 加权平均资本成本法（WACC）**

```
WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-Tc)
```
- `Re`：股权成本（通过 CAPM 计算）
- `Rd`：债务成本（税前）
- `Tc`：企业所得税率
- `E`：股权市值
- `D`：债务市值

**2.2 资本资产定价模型（CAPM）**

```
Re = Rf + β × (Rm - Rf) + α
```
- `Rf`：无风险利率（通常取 10 年期国债收益率）
- `β`：行业贝塔系数
- `Rm`：市场平均回报率
- `α`：数据资产特有风险溢价（通常 1%-5%）

**2.3 风险调整**

| 风险因素 | 调整方向 | 典型调整幅度 |
|---------|---------|------------|
| 数据时效性风险 | 上调折现率 | +0.5%-2% |
| 技术迭代风险 | 上调折现率 | +0.5%-3% |
| 法律合规风险 | 上调折现率 | +0.5%-2% |
| 市场竞争风险 | 上调折现率 | +0.5%-2% |
| 数据安全风险 | 上调折现率 | +0.5%-1.5% |

### 第三步：估值计算

**折现公式：**
```
V = Σ [CFt / (1+r)^t] + TV / (1+r)^n

TV = CFn × (1+g) / (r-g)   （永续增长法）
```
- `V`：数据资产评估值
- `CFt`：第 t 年净现金流
- `r`：折现率
- `TV`：终值
- `g`：永续增长率（通常取 0%-3%）
- `n`：预测期

### 第四步：敏感性分析

| 敏感性变量 | 变化幅度 | 估值影响 | 分析方式 |
|-----------|---------|---------|---------|
| 折现率 | ±1% | 估值变化 X% | 单变量敏感分析 |
| 收入预测 | ±10% | 估值变化 X% | 单变量敏感分析 |
| 预测期限 | ±1年 | 估值变化 X% | 单变量敏感分析 |
| 永续增长率 | ±0.5% | 估值变化 X% | 单变量敏感分析 |
| 组合分析 | — | — | 双变量敏感性矩阵 |

### 第五步：适用性判断

**收益法适用条件（须同时满足）：**

| 序号 | 适用条件 | 判定标准 |
|------|---------|---------|
| 1 | 有明确商业化路径 | 已有或可预见的产品/服务/交易模式 |
| 2 | 未来收益可预测 | 有历史数据或可比案例支撑预测 |
| 3 | 风险可量化 | 关键风险因素可识别并量化 |
| 4 | 预测期合理 | 可合理设定 3-5 年预测期 + 终值 |

## Steps

1. **读取输入参数**：资产名称、预测期、收入/成本预测、折现率参数
2. **适用性判断**：检查收益法四项适用条件
3. **现金流预测**：逐年计算收入、成本、净现金流
4. **折现率计算**：若未提供，按 WACC/CAPM 计算 + 风险调整
5. **终值计算**：永续增长法计算终值
6. **折现计算**：逐年折现并求和 + 终值折现
7. **敏感性分析**：对折现率±1%、收入±10%、期限±1年进行分析
8. **输出估值结论**：基准估值 + 敏感性区间
9. **风险提示**：列示关键假设与风险因素
10. **标准溯源**：附 `§7.2.X` 标注

## 常见问法与处理策略

| 常见问法 | 处理策略 | 输出要点 |
|---------|---------|---------|
| "用收益法算一下这批数据值多少" | 检查适用性→执行完整 DCF | 估值结论 + 现金流表 + 敏感性 |
| "折现率怎么选？" | 按 WACC/CAPM 计算 + 风险调整 | 折现率 + 计算过程 + 调整说明 |
| "收入预测怎么做的？" | 追问商业化路径，分场景预测 | 收入预测表 + 假设说明 |
| "收益法适用这批数据吗？" | 执行四项适用性判断 | 适用/不适用 + 原因 |
| "折现率变动1%影响多大？" | 执行折现率敏感性分析 | ±1% 估值变化 |
| "终值怎么算？" | 永续增长法 | 终值计算 + 增长率依据 |
| "和成本法差异大怎么办？" | 分析差异原因，建议加权 | 差异分析 + 加权建议 |
